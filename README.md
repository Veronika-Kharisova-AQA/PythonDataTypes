# Задание
1. Создать словаря email
2. Добавить даты отправки 
3. Нормализовать e-mail адрес отправителя и получателя: привести к нижнему регистру и уберать пробелы по краям.
4. Извлечь логин и домен отправителя в две переменные login и domain
5. Создать сокращённую версию текста
6. Создать список личных доменов
7. Проверить что в списке личных и корпоративных доменов нет пересечений
8. Проверить «корпоративность» отправителя
9. Собрать «чистый» текст сообщения 
10. Сформировать текст отправленного письма многострочной f-строкой и сохранить в email
11. Рассчитать количество страниц печати для email
12. Проверить пустоту темы и тела письма
13. Создать «маску» e-mail отправителя 
14. Удалить из списка личных доменов значения "list.ru" и "bk.ru".
